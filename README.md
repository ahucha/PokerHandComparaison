# Texas Hold'em Poker - Hand Comparison

Ce projet implémente un évaluateur de mains de poker Texas Hold'em en suivant la méthodologie **TDD**. Le programme détermine la meilleure main de 5 cartes parmi 7 et compare les résultats entre plusieurs joueurs.

## 🚀 Installation et Tests

Le projet utilise **Python 3** et le framework **pytest**.
